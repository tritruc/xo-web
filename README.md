# XO Infinite (Production rebuild)

Web game XO/Caro mobile-first, board infinite, deploy free on GitHub Pages.

## Features
- Infinite board (pan/zoom)
- PvE (AI heuristic)
- PvP room (PeerJS host/join by room number)
- Sound effects + victory status

## Run locally
Open `index.html` with a static server.

## Deploy
GitHub Pages from `main` branch root.
